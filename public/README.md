# Brand Generator

An AI-powered brand identity generator with visual taxonomy controls.

## Features
- Multi-output brand asset generation
- Color palette management  
- Visual taxonomy slider controls
- Style preset system

## Local Development

To run this locally:

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000
